2.請找出三個課程沒教的 HTTP status code 並簡單介紹

201 : 請求成功且新的資源成功被創建，通常用於 POST 或一些 PUT 請求後的回應。

403 : 用戶端無訪問權限，所以伺服器拒絕給予應有個回應。

414 : 客戶端的 URI 請求超過伺服器願意解析的長度。
